####1️⃣ What is the difference between var, let, and const?
- **var: Function scoped re-declared and updated anytime.**
- **let: Block scoped can be update but not re-declared.**
- **const: Block scoped cannot be updated or re-declared.**

####2️⃣ What is the spread operator (...)?
- Ans: **It allow copy all part of an existing array or object into another array or object easily.**

####3️⃣ What is the difference between map(), filter(), and forEach()?
- **map: Returns a new array with modified data.**
- **filter: Returns a new array with elements that meet a condition.**
- **forEach: Just loops through the array don't return anything.**

####4️⃣ What is an arrow function?
- Ans: **A modern (es6) way to write functions. Syntax () => ;**

####5️⃣ What are template literals?
- Ans: **Strings defined with backticks ``. that allow dynamic string ${}**
